# icicle
EN (Repository Description, short) icicle is a Windows-first disk cleanup toolkit: fast size tree, top heavy files, smart folder watch auto-sort, and one-click GUI actions.  RU (Краткое описание репозитория) icicle — Windows-first комбайн для чистки диска: быстрый size tree, топ тяжёлых файлов, авто-сортировка новых файлов и удобный GUI.
