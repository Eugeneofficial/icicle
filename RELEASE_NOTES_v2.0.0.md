# Release Notes v2.0.0

## EN

icicle 2.0.0 introduces a native desktop app (Wails), fast heavy-file workflows, compact file actions, and improved release/update tooling.

### Highlights
- Native desktop app (`icicle-desktop.exe`) without browser dependency
- Fast heavy mode with file-cap and worker tuning
- New compact heavy actions panel (open/reveal/move/delete/undo)
- Export heavy results to CSV / JSON / Markdown
- Tray integration + reopen
- In-app update flow improvements
- Full docs refresh: README, roadmap, visuals

## RU

В версии 2.0.0 проект получил нативное desktop-приложение, более быстрый workflow для тяжёлых файлов и полный апгрейд документации/релизного контура.

### Основное
- Нативный desktop (`icicle-desktop.exe`) без браузера
- Быстрый режим heavy с настройкой лимита файлов и потоков
- Новый компактный блок действий по файлам
- Экспорт heavy-результатов в CSV / JSON / Markdown
- Трей-иконка и быстрое повторное открытие окна
- Улучшения сценария обновлений
- Полная переработка README/ROADMAP и визуалов
